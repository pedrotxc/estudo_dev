# estudo_design_patterns_em_java

Padrăo de projeto: "Soluçăo comum para um problema recorrente ao utilizar o paradigma da orientaçăo a objetos".  

Desing Patterns - Categorias  
.Criacionais  
.Estruturais  
.Comportamentais  

-Adapter é implementdo com a criaçăo de uma classe adaptadora qeu atua como uma ponte entre o cliente e o objeto com a interface incompatível. Existem duas abordagens comuns para implementar o padrăo Adapter em Java: A adaptaçăo de objeto e a adaptaçăo de classe.

-Decorator é implementado com a criaçăo de classes que estendem uma interface comum ou uma classe abstrata, permitindo adicionar funcionalidades extras de forma dinâmica a um objeto.
